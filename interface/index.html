About Resume'Bot

Resume'Bot is a Raspberry Pi powered robot, that interfaces with a small remote controlled car.

The Pi interfaces with the car with an AdaFruit 16 channel PWM board, and uses the following software technologes:

  • Node.Js
  • Express.Js
  • Socket.IO
  • JavaScript
  • Python
  • HTML
  • BootStrap
  • JQuery
